# Standoff 2 Dumper

[![Build](https://github.com/FANATFANATA/Standoff2-Dumper/actions/workflows/build.yml/badge.svg?branch=master)](https://github.com/FANATFANATA/Standoff2-Dumper/actions/workflows/build.yml)

## Особенности

- Нет необходимости в global-metadata.dat

## Требования

- Android устройство с root
- Игра Standoff 2

## Использование

1. Переместите собранный дампер в /data/local/tmp/ на вашем устройстве
2. Запустите собранный дампер командой su -c "./Standoff2-Dumper" из консоли или с помощью MT Manager, выбрав в нем запуск с рут правами.
3. Готовый dump.cs будет сгенерирован в /storage/emulated/Android/obb/com.axlebolt.standoff2/dump.cs
